**Generalización**

Es agrupar varias entidades parecidas en una más general.

Piensa en: Perro, Gato, Ave → se generalizan en Animal.

La entidad general contiene lo común a todas.

**Especialización**

Es lo contrario: partir de una entidad general para crear entidades más específicas.

Ejemplo: Vehículo → Coche, Moto, Camión.

Cada entidad específica tiene atributos propios además de los generales.

**¿Para qué sirve?**

Para organizar mejor la información cuando hay entidades que comparten cosas pero también tienen diferencias.

Evita duplicar atributos y hace la base de datos más clara.
