En Python, las listas son estructuras muy útiles porque permiten almacenar varios elementos dentro de una sola variable.
El propio lenguaje Python nos ofrece un método llamado sort(), que sirve para ordenar los elementos de una lista en orden alfabético o numérico.

Primero, defino una lista con tres frutas: 'platano', 'fresa', 'naranja'.
Después, aplico el método sort(), que ordena directamente los elementos de la lista.


**Código:**


frutas = 'platano', 'fresa', 'naranja'   (Lista original)
frutas.sort()   (Ordenamos la lista con el método sort ) 

print("Lista de frutas ordenada:", frutas)


El resultado del anterior código es el siguiente:
Lista de frutas ordenada: ['fresa', 'naranja', 'platano']

El método sort() ha ordenado los elementos de la lista en orden alfabético ascendente, tal y como se esperaba.


En conclusión, el método sort() es una herramienta sencilla pero muy útil para ordenar listas en Python.
