Las variables en programación son espacios de almacenamiento a los que se les asigna un **nombre** para guardar y representar diferentes tipos de **datos o valores**

Piensa en una variable como una caja etiquetada, la etiqueta es el nombre de la variable.

Vamos a declarar dos variables en este ejercicio, una sera:
**Nombre:** nombre_atleta
**Edad:** edad_atleta


Vamos a asignar a cada variable un nombre y una edad, en este caso el del atleta:
**nombre_atleta** = "Javier Carrasco"
**edad_atleta** = "28"


La función principal del comando "print" es mostrar datos o resultados en la consola o pantalla para que el usuario o el desarrollador pueda verlos.

En este caso, teclearemos:
print("nombre_atleta")
print("edad_atleta")


En resumen, las variables son el mecanismo que enseña al software de la capacidad de interactuar con datos cambiantes y realizar tareas generales sin necesidad de volver a escribirlo de nuevo.
