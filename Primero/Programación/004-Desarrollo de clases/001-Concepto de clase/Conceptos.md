Las funciones en Python permiten agrupar instrucciones para realizar una tarea específica y reutilizar el código fácilmente.
En este ejercicio voy a crear una función llamada saluda que reciba dos parámetros: el nombre y la edad del nadador.

Uso la palabra clave def para definirla.

Especifico los parámetros nombre y edad dentro de los paréntesis.

En el interior de la función, utilizo print() para mostrar un mensaje que combine texto y variables.

**Código**:


def saluda(nombre, edad):
    print("Hola", nombre + ", tienes", edad, "años. ¡Bienvenido a la piscina!")


#### Llamadas a la función con distintos valores
saluda("Javier", 21)
saluda("Lucía", 17)
saluda("Carlos", 25)


Cada vez que se llama a la función, se pasan valores diferentes para los parámetros nombre y edad.
En este ejercicio he aprendido cómo definir y utilizar una función con parámetros en Python.
