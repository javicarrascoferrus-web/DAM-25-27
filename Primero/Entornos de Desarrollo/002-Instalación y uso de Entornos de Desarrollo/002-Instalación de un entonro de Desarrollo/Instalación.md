Visual Studio Code (VSCode) es un editor de código multiplataforma creado por Microsoft que se ha convertido en una de las herramientas más populares entre los programadores.
En esta actividad, el objetivo es descargar, instalar y configurar VSCode, y después hacer una pequeña prueba ejecutando un programa en Python para confirmar que funciona correctamente.

**Paso 1: Descarga de VSCode**

Accedí a la página oficial

Seleccioné el instalador correspondiente a mi sistema operativo (en mi caso, Windows 11 de 64 bits).

Esperé a que finalizara la descarga del archivo .exe.


**Paso 2: Instalación**

Ejecuté el instalador descargado (VSCodeSetup-x64.exe).

Acepté los términos de licencia y mantuve las opciones por defecto.

Hice clic en “Instalar” y esperé a que completara el proceso.

Una vez finalizada la instalación, abrí Visual Studio Code.


**Paso 3: Configuración del entorno**

Cclické dos veces en Python (para ejecutar código Python directamente).

Spanish Language Pack (para tener el entorno en español).

Prettier (para formatear automáticamente el código).


Con esta práctica he aprendido a descargar, instalar y configurar Visual Studio Code, así como a usarlo para escribir y ejecutar código Python.
Esta actividad se relaciona con los contenidos de la unidad sobre instalación y configuración de entornos de programación.
