A medida que avanzo en mi formación como programador, voy descubriendo que un entorno de desarrollo no solo sirve para escribir código, sino también para personalizar la experiencia de trabajo y hacerla más eficiente.
Visual Studio ofrece una enorme cantidad de complementos (extensiones) que permiten adaptar el entorno a las necesidades de cada desarrollador.

**Paso 1: Abrir Visual Studio**
Abrí Visual Studio desde el menú de inicio. Una vez cargado el entorno, esperé a que apareciera la pantalla principal con mis proyectos recientes y el panel lateral.

**Paso 2: Buscar complementos**

Fui a la barra superior y seleccioné:
Extensiones → Administrar extensiones.

Se abrió una ventana con una tienda de complementos organizados por categorías: productividad, depuración, apariencia, etc.

En el cuadro de búsqueda escribí “Theme” para buscar temas visuales que cambian el aspecto del editor.

**Paso 3: Seleccionar e instalar un complemento**
Elegí el complemento “One Dark Pro Theme”, un tema de colores inspirado en Visual Studio Code, que mejora el contraste y hace el código más legible.

Hice clic en “Descargar”.

Visual Studio me pidió reiniciar para completar la instalación.

Tras reiniciar, el complemento apareció disponible en la lista de temas.


**Comprobación del complemento instalado**
Una vez reiniciado Visual Studio:

Fui a Herramientas < Opciones < Entorno < General.

En la sección Tema de color, seleccioné One Dark Pro.

El editor cambió a un fondo oscuro con colores vivos para las palabras clave, comentarios y cadenas de texto.



Con esta práctica aprendí a instalar y configurar complementos en Visual Studio, comprendiendo que estas herramientas son clave para adaptar el entorno a las necesidades de cada desarrollador.
