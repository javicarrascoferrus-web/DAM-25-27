En esta actividad el objetivo es instalar Python, preparar el entorno en Visual Studio Code (VSCode) y ejecutar mi primer programa.
Python es un lenguaje de programación muy popular por su sencillez y potencia. Instalarlo y configurarlo correctamente es el primer paso para empezar a programar.

**Paso 1: Instalar Python**

Entré en la página oficial de Python

Descargué la versión recomendada para Windows (Python 3.x).

Ejecuté el instalador y marqué la opción “Add Python to PATH” para poder usarlo desde la terminal.


**Paso 2: Abrir VSCode**

Abrí Visual Studio Code desde el menú de inicio.

Verifiqué que tenía instalada la extensión de Python, que es necesaria para ejecutar el código dentro del editor.


**Paso 3: Crear un nuevo archivo**

En VSCode seleccioné Archivo → Nuevo archivo.

Lo guardé con el nombre primeraprueba.py dentro de una carpeta de trabajo (por ejemplo, ProyectosPython).
