En esta práctica he aprendido a crear un proyecto en Visual Studio y a ejecutar un programa en C#, uno de los lenguajes más usados en el desarrollo de aplicaciones de escritorio y empresariales.
El objetivo era crear una aplicación de consola llamada “NatacionApp”, escribir un pequeño código que muestre mensajes en pantalla.

**Paso 1: Crear el proyecto en Visual Studio**

Abrí Visual Studio desde el menú de inicio.

En la pantalla de inicio seleccioné la opción “Crear nuevo proyecto”.

En el buscador de plantillas elegí “Aplicación de consola (Console App)” con el lenguaje C#.

Asigné el nombre NatacionApp al proyecto.

Seleccioné la ubicación donde guardar los archivos y confirmé.

**Paso 2: Escribir el código fuente**
Una vez creado el proyecto, Visual Studio generó automáticamente el archivo Program.cs, donde escribí el siguiente código:
using System;

namespace NatacionApp
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("¡Bienvenido a la natación!");
            Console.WriteLine("Aprende nuevos trucos cada día.");
            Console.ReadLine();
        }
    }
}


**Paso 3: Compilar y ejecutar**

Pulsé el botón Play o presioné F5 para compilar y ejecutar el proyecto.

Visual Studio compiló el código y abrió una ventana de consola.

En la ventana apareció el mensaje:

¡Bienvenido a la natación!
Aprende nuevos trucos cada día.

Este pequeño programa demuestra cómo Visual Studio crea automáticamente la estructura básica de una aplicación en C# con namespace, class y Main


Con esta práctica aprendí a crear, escribir, compilar y ejecutar un programa en C# dentro de Visual Studio, comprendiendo mejor cómo funciona un entorno de desarrollo completo (IDE).
