Para hacer los ejercicios del ciclo estoy usando Gedit, el editor de texto por defecto del escritorio GNOME. Me viene bien porque es ligero, multiplataforma y tiene resaltado de sintaxis para varios lenguajes (Python, HTML, CSS, etc.).

Trabajo en Windows, pero utilizo este editor de texto porque Visual Estudio tiene muchos pitos y flautas y acaba mareando, muy ocurito y tal pero de momento es demasiado para lo que necesito.

Se descarga en la plataforma oficial de Microsoft, con un coste de 3€. El protocolo de  instalacion de Windows es sencillo, se pueden realizar mediante comandos tambien pero por interfaz gráfica es más rapido y cómodo.


Con Gedit tengo un entorno ligero y suficiente para los objetivos de la unidad: edición con resaltado, control de sangrías, plugins básicos y una forma rápida de ejecutar scripts.

Si más adelante necesito depuración avanzada o integración con entornos complejos, podría migrar a un IDE más “grande”, pero para el ciclo, Gedit + plugins me permite trabajar rápido, ordenado y sin distracciones.
