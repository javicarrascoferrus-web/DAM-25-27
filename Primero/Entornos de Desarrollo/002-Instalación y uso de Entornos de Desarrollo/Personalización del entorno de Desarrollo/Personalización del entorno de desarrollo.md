En programación, pasar horas frente al código es algo habitual, por eso es importante tener un entorno de desarrollo cómodo y personalizado. En esta actividad voy a realizar algunos ajustes básicos en Visual Studio Code (VSCode) para adaptarlo a mis gustos y mejorar la legibilidad del código.

**Paso 1: Abrir el entorno de desarrollo**

Abrí Visual Studio Code desde el menú de inicio.

Esperé a que cargara la pantalla principal del editor.


**Paso 2: Cambiar el color del tema**

Fui a la barra superior y seleccioné:
Archivo → Preferencias → Tema de color.

VSCode muestra una lista de temas preinstalados como: Dark+, Light+, Monokai, Solarized Dark, etc.

Probé varios temas hasta encontrar uno que me resultara más cómodo para la vista.
En mi caso, elegí “Dark+ (predeterminado oscuro)” porque ofrece buen contraste y es menos cansado para la vista.


Después de cambiar el tema y el tamaño de texto, abrí un archivo de prueba llamado ejemplo.py con el siguiente contenido:

mensaje = "Probando configuración de VSCode"
print(mensaje)

Antes de los cambios, las letras se veían pequeñas y el tema claro me resultaba algo molesto.
Tras aplicar el tema oscuro y aumentar el zoom, el código se veía mucho más nítido y cómodo.

Con esta práctica aprendí a personalizar Visual Studio Code para trabajar de forma más cómoda.
Cambiar el tema de color y ajustar el tamaño del texto puede parecer algo menor, pero en realidad mejora mucho la experiencia del programador y la productividad diaria.
