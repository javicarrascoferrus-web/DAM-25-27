En esta práctica trabajamos el concepto de encapsulación de funciones en Python, una técnica que permite organizar el código y reutilizarlo fácilmente.
El objetivo es crear una función llamada division que reciba dos parámetros (operando1 y operando2), realice la operación de división dentro de la función y devuelva el resultado.

El ejercicio se resuelve creando una función en un archivo llamado encapsulacion_a_funcion.py, siguiendo los pasos indicados:

**Paso 1: Definir la función**

La función se llama division.

Acepta dos parámetros: operando1 y operando2.

Dentro de la función, se ejecuta la operación de división y se devuelve el resultado.

**Paso 2: Probar la función**

En el bloque principal del código (if __name__ == "__main__":), se llama a la función division(4, 3) y se muestra el resultado con print().

Con esta práctica comprendí cómo encapsular operaciones dentro de funciones, separando la lógica de cálculo del código principal.
Este tipo de estructura hace que el código sea más claro, modular y fácil de depurar.
