En esta unidad hemos trabajado con distintos lenguajes que se usan en el desarrollo y la programación de aplicaciones o páginas web. Concretamente, hemos visto **Python, HTML y CSS.**
Cada uno tiene una función diferente dentro del mundo de la informática: mientras Python se usa como lenguaje de programación, HTML y CSS se emplean para estructurar y diseñar sitios web.

Un ejemplo práctico de cómo pueden interactuar los tres lenguajes sería el desarrollo de un pequeño sitio web con una parte de programación en Python (por ejemplo, con el framework Flask), donde:

**HTML** define la estructura de la página.

**CSS** aplica los estilos visuales.

**Python** maneja la lógica y la interacción con el servidor.

Durante la unidad hemos aprendido que:

**HTML y CSS** son lenguajes de marcado y estilo, enfocados en la parte visual y estructural del desarrollo web.

**Python**, en cambio, es un lenguaje de programación interpretado, con sintaxis sencilla y multiplataforma, ideal para aprender lógica de programación, estructuras de control y manipulación de datos.
