El procesador es el componente físico del ordenador encargado de ejecutar las instrucciones de los programas. Cada programa informático genera uno o varios procesos, que son las tareas que el procesador toma y ejecuta.

Cuando ejecutamos Python, escribimos el siguiente código:
print("Hola mundo")

Cuando presionamos enter, a continuación el programa dirá :"Hola mundo"

Este mensaje confirma que el procesador ha ejecutado correctamente las instrucciones contenidas en el archivo de Python.

Tambien se puede realizar escribiendo en un documento de texto y guardandolo como .py, nunca como .txt

Siempre hay que estar en el directorio donde se guarda el archivo .py para poder ejecutarlo

Este ejercicio nos ha permitido comprender cómo un programa informático genera un proceso que se ejecuta en el procesador, mostrando un resultado en pantalla.
