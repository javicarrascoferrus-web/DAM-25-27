Objetivo: versionar el proyecto de la asignatura con Git + GitHub y manejar el flujo básico: repo remoto < clonado local < commits < push < ver histórico.

**Crear cuenta en GitHub**

Entra en github.com y crea la cuent

Verifica el email 


**Crear repositorio remoto**

Botón New (arriba a la izquierda).

Nombre: gestor-clientes

Visibilidad: Private o Public.

No marques plantilla si ya tienes proyecto local

Create repository.

**Clonar con GitHub Desktop**

Abre GitHub Desktop < File > Clone repository…

Pestaña GitHub.com < selecciona tu repo < Clone.

Elige carpeta local

Nota: Si ya tienes el proyecto en otra carpeta, mueve tus archivos a la carpeta clonada para evitar mezclar repos.

**Poner el código del proyecto en la carpeta clonada**

Copia los .py del proyecto a la carpeta clonada.

Vuelve a GitHub Desktop: verás los cambios en Changes.

**Primer commit y push**

En Summary, escribe un mensaje: feat: añade versión inicial del gestor de clientes.

Commit to main.

Push origin (botón arriba).

Comprueba en github.com (repo > pestaña Code) que los archivos están.


He completado el flujo esencial: crear repo < clonar < commit < push < verificar histórico y una modificación posterior con su commit.
