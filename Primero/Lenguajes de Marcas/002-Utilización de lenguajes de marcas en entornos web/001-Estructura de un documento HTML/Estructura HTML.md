En este ejercicio se busca entender la estructura básica de una página web en HTML, que es el lenguaje estándar para crear sitios web.
Cada documento HTML tiene una estructura jerárquica compuesta por el doctype, la etiqueta raíz <html>, una sección de encabezado <head> y otra de contenido <body>

**Los elementos principales que se deben incluir son:**

<!doctype html> → indica que el documento está escrito en HTML5.

<html lang="es"> → abre el documento y especifica el idioma (en este caso, español).

<head> → sección para información del documento.

<title> → define el título que aparecerá en la pestaña del navegador.
	
	
**A continuación se muestra un ejemplo básico completo del archivo index.html:**

<!doctype html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Página de inicio de José</title>
</head>
<body>
  <h1>Bienvenido a la página de José</h1>
  <p>Esta es mi primera página web creada con la estructura básica de HTML.</p>
</body>
</html>
	
	
Con esta práctica, he aprendido la estructura mínima que debe tener cualquier documento HTML y la importancia de definir correctamente su encabezado.
El uso del doctype, las etiquetas <html>, <head> y <title> permite que la página sea compatible con todos los navegadores.
