En este ejercicio vamos a practicar el código HTML y el SVG, con un ejercicio práctico y sencillo.
Para ello vamos a empezar tecleando el código de entrada y añadiendo el código SVG, quedaría tal que así.

<!doctype html>
<html>
  <head>
    <title>Gráfico SVG</title>
  </head>
  <body>
<svg width="512" height="512" xmlns="http://www.w3.org/2000/svg">
  <circle cx="256" cy="256" r="100" fill="red"/>
	<circle cx="100" cy="256" r="50" fill="blue"/>
<circle cx="400" cy="256" r="75" fill="green"/>
</svg>
  </body>
</html>

Una vez escrito el código, podemos comprobar que los circulos añadidos hacen del gráfico mas interesante, más moderno y mucho más aseado.
Los circulos sirven para darle colorm tamaño y apariencia a ciertos entornos de la web.
Este tipo de código se usa comúnmente en páginas web para mostrar gráficos que pueden escalar a cualquier tamaño sin perder calidad.

Por último, guardaremos el archivo en .svg.html para poder visualizar en de forma correcta en un navegador web.
