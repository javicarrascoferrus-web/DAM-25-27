En entornos de programación, es primordial utilizar editores de texto en los que teclear para escribir código.
En clase, el profesor ha recomendado Gedit, no solo porque utiliza Linux, si no porque es bastante sencillo y claro de utilizar.

Cuando tecleamos código, ya sea en el editor que sea, si estamos trabajando con lenguaje HTML, lo guardaremos siempre con .html, si es Python .py si es CSS .css , etc...
En este ejercicio vamos a instalar Gedit, para sistemas Linux es gratuito, pero para sistemas Windowas y Mac es de pago, en este caso vale poco menos de 3€
Una vez comprado he instalado por la tienda oficial de Microsoft, nos disponemos a abrirlo y teclear código.
Para ello, como bien dice el ejercicio, teclearemos lo siguiente:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi primera página web</title>
</head>
<body>
    <h1>Bienvenido a mi sitio web</h1>
    <p>Este es un párrafo de texto.</p>
</body>
</html>

Este código deberemos guardarlo como "nombre del archivo".html (nombrando como queramos) y guardarndolo en un directorio a mano en el PC.

En Gedit es sencillo de teclear, modificar y guardar código que estemos trabajando. Es un editor que puede ser utilizado tanto como para principiantes o para avanzados
